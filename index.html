Connect a Google Apps Script web app URL to sync all data live with Google Sheets. All entries will be saved there and shared with your team.
Apps Script Web App URL
How to set up Google Sheets sync:
1. Open your Google Sheet → Extensions → Apps Script
2. Paste the provided script code → Save
3. Deploy → New deployment → Web app → Anyone → Deploy
4. Copy the Web app URL and paste it above
